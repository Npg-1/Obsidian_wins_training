## URI ##
URI(Uniform Resource Identifier)는 인터넷상의 리소스 "자원 자체"를 식별하는 문자열임

## URL ##
URL(Uniform Resource Locator)는 인터넷상의 리소스 자원 자체의 **위치**를 식별하는 문자열로 URI와 위치를 동시에 보여줌
URL은 웹 사이트 주소뿐만 아니라 컴퓨터 네트워크 상의 자원을 모두 나타내는 표기법으로 특정 웹  사이트에 접속하기 위해서는 해당 휍 사이트의 주소뿐만 아니라 프로토콜(https, http, stfp 등)을 함께 알아야 접속이 가능함, 이들 모두를 나타내는 게 URL임

## URI와 URL의 차이 ##
URI = 식별자,  URL = 식별자 + 위치
예를 들어서 elancer.co.kr은 URI임 리소스의 이름만 나타냄 하지만 https\://elancer.co.kr는 URL임 이름과 위치까지 나타내기 때문(프로토콜인 https 포함)
따라서 URI가 URL보다 더 큰 개념임
