# 개발자 도구 #

\* *여기서는 크롬을 기준으로 설명함* \*

개발자 도구는 브라우저 화면에서 F12를 누르거나, 우클릭 -> 검사를 통해서 들어갈 수 있음

![[Pasted image 20260118114054.png]]


### 개발자 도구 ###

| 색상      | 설명                                                                                                                                                                                                                                                                                                                                                 |
| ------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **빨간색** | 요소 검사(Inspect) 및 디바이스 툴바(Device Toolbar)                                                                                                                                                                                                                                                                                                           |
| **주황색** | 기능을 선택하는 패널. 여기선 **굵게**처리된 기능들만 다룸<br>- **Elements**: 페이지를 구성하는 HTML 검사<br>- **Console**: 자바 스크립트를 실행하고 결과를 확인할 수 있음<br>- **Sources**: HTML, CSS, JS 등 페이지를 구성하는 리소스를 확인하고 디버깅할 수 있음<br>- **Network**: 서버와 오가는 데이터를 확인할 수 있음<br>- Performance<br>- Memory<br>- **Application**: 쿠키를 포함하여 웹 어플리케이션과 관련된 데이터를 확인할 수 있음<br>- Security<br>- Lighthouse |
| **노란색** | 현재 페이지에서 발생한 에러 및 경고 메시지                                                                                                                                                                                                                                                                                                                           |
| **녹색**  | 개발자 도구 설정                                                                                                                                                                                                                                                                                                                                          |

# Console #
콘솔은 프론트엔드의 자바스크립트 코드에서 발생한 각종 메시지를 출력하고, 사용자가 입력한 자바스크립트 코드를 실행도 할 수 있는 도구임

자바스크립트로 웹 개발을 할 때, Console 오브젝트에는 개발자 도구의 콘솔에 접근할 수 있는 함수가 정의되어 있음. 코드를 작성하면서 어떤 변수의 값을 중간에 출력해보고 싶다면, console.log 등의 유용하게 사용할 수 있음

자바스크립트 코드를 잘 모르니까 일단 PASS

# Network #

브라우저가 서버와 오가는 데이터를 확인할 수 있음
![[Pasted image 20260118161713.png]]


| 색상      | 설명                                                                                         |
| ------- | ------------------------------------------------------------------------------------------ |
| 빨간색     | 로깅 중지 및 로그 전체 삭제                                                                           |
| **주황색** | 로그 필터링 및 검색                                                                                |
| **녹색**  | 옵션<br>- Preserve log: 새로운 페이지로 이동해도 로그를 삭제하지 않습니다.<br>- Desable cache: 이미 캐시된 리소스도 서버에 요청함 |
| **노란색** | 네트워크 로그                                                                                    |
| **파란색** | 네트워크 로그 요약 정보                                                                              |
