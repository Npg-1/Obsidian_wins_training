- ## DHCP(Dynamic Host Configuration Protocol) ##
	- 클라이언트에 대한 네트워크 설정이나 IP 주소 할당을 자동화하기 위한 프로토콜
	- 네트워크상의 각 클라이언트는 DHCP 서버에 요청하여 자신이 사용할 IP 주소를 할당받아 네트워크에 연결할 수 있음
	- 할당받은 IP 주소는 TCP/IP 네트워크에서 각 컴퓨터를 식별하기 위해 사용되므로 각 컴퓨터는 서로 중복되지 않는 주소를 할당받아 사용해야함
- ## DNS(Domain Name System) ##
	- IP 주소와 컴퓨터 이름을 관리하는 서비스
	- DNS를 통해 컴퓨터 이름(예: naver.com)으로 IP 주소를 알아내 접근할 수 있으므로, IP주소를 몰라도 해당 컴퓨터에 접근할 수 있음
- ## 게이트웨이 ##
	- 게이트웨이는 로컬 네트워크와 외부 네트워크를 연결하는 출입문이라 볼 수 있다.
	- 예를 들어 일반적인 가정에서는 공유기가 이 게이트웨이 역할을 수행함
