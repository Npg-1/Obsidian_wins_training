
TMS-PLUS란?
자사 보안장비의 이벤트와 네트워크 정보를 수집/분석/대응하고 **연동장비를 관리**하는 통합보안관리시스템
- NGFW, DDoS, IPS, APT 등의 로그를 다 받을 수 있음


상황판에서 장비의 현재 상태, 로그들을 볼 수 있음

TMS는 데이터 로그를 많이 보관하기 때문에 하드디스크와 메모리 용량이 많이 큼

Snort 배포는 -rule을 배포해서 각 장비에 뿌리는 기능을 가지고 있음

## 2.2 모니터링 ##
연동된 센서와 등록된 객체에서 전송된 이벤트와 네트워크 트래픽을 제공하는 메뉴

- 상황판 - 실시간 서버현황, 이벤트/트래픽 현황, 분석 등 레이아웃 별 표시
- 이벤트 모니터링 - 실시간 탐지되는 이벤트에 대한 정보를 확인할 수 있는 메뉴
	  - 필터 뷰: 기본 필터/고급 필터 적용으로 조건에 해당되는 이벤트만 모니터링
	  - 데이터 연관분석: 탐지된 이벤트에 대한 연관분석 정보를 확인
	  - IP/PORT 차단: 탐지된 이벤트에 대한 IP/PORT 차단
  - 트래픽: 연동된 장비 별 실시간 트래픽을 확인하는 메뉴
  - 실시간 트래픽: 연동된 장비 별 실시간 INLINE 트래픽을 확인하는 메뉴(트래픽이랑 차이점이 INLINE 밖에 없는 건가)
  - 세션: 연동된 방화벽 센서의 세션 모니터링 확인하는 메뉴
